# 9anime Extension

## Installation
1. Clone the repository:
   ```bash
   git clone https://github.com/IzzatHasboloah/Repo-seanime.git
   ```
2. Navigate to the project directory:
   ```bash
   cd Repo-seanime
   ```
3. Install dependencies:
   ```bash
   npm install
   ```

## Usage
- To start the extension, run:
  ```bash
  npm start
  ```
- Open your browser and navigate to 9anime to use the extension features.
